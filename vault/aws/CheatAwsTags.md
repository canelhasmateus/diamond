# TemplateAwsTags

While using [[IndexAws]], Tags can be used in a variety of ways:

* Technical Tags:
  * Name
  * Application Id
  * Application Role
  * Cluster
  * Environment
  * Version
* Automation:
  * Date/Time,
  * Opt-in/Opt-out,
  * Security
* Business:
  * Owner,
  * Cost Center
  * Business unit,
  * Customer,
  * Project
  * security:
  * Confidentiality,
  * Compliance
