# Hashing

Hashing is a process in which a value of any kind is deterministically transformed into a number or a fixed-length string.

Applications:

* [[Caching]]
* [[HashTables]]
* [[Sharding]]
* [[Partitioning]]
* [[SoftwareSecurity]], due to usually being an irreversible process
